# Changelog

## Phase 1 - Detection Foundation
- Built the telemetry ingestion path for industrial sensor data.
- Added the Random Forest anomaly detector and baseline device/status models.

## Phase 2 - Agentic Risk Pipeline
- Added the LangGraph pipeline for detection, classification, isolation, and financial quantification.
- Introduced severity-based downtime, SLA, and regulatory penalty calculations.

## Phase 3 - Reporting and Operator Visibility
- Generated incident summaries and credit-risk briefs for downstream reporting.
- Added the live dashboard, API endpoints, and simulator workflow for end-to-end demos.
